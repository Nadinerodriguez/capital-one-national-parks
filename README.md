# Capital One Software Engineering Summit (Summer 2019)

## Description

**Challenge:** Build a web app that serves as a National Park Service information kiosk

**Link:** https://www.mindsumo.com/contests/national-park-api

**Link to the website I created:** https://www.nadinebrodriguez.com/

## Tech Stack

### Backend:
- Django
- National Park's API

### FrontEnd:
- HTML
- CSS
- JavaScript
- jQuery
- Bootstrap

## Features

### Required:
- [ ] Filtering
    - State
    - Designations
- [ ] Searching
    - Name
    - Keyword
- [ ] Destination
    - Alerts
    - Articles
    - Events
    - News Release
- [ ] Detail
    - Visitor centers
    - Campgrounds
    - Lesson plans
    - People

### Stretch:
- [ ] Map Visualizations
- [ ] NPS Relevant Symbols
